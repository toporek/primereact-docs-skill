# Add-Ons

Premium templates, UI blocks and design tooling that build on top of PrimeReact.

## Overview

Add-Ons are premium, production-ready resources built on top of PrimeReact. They help you skip the boilerplate and ship faster, from full application templates to ready-made UI blocks and design tooling.

## What's included

- **Templates**, complete, themeable application starters with layouts, pages and integrations wired up.
- **Blocks**, copy-and-paste UI sections for application, marketing and e-commerce pages.
- **Figma UI Kit**, the design source that mirrors the component library, for designing before you build.
- **Theme Designer**, a visual tool to craft and export your own design tokens and themes.

Pick a category from the sidebar to explore each add-on.
